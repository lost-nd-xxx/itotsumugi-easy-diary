# ひとこま — ローカル日記 PWA

一日のかけらを、アイコンで簡易に綴る日記アプリです。
スマホ（Android Chrome）で「ホーム画面に追加」すると、アプリのように使えます。

- 完全無料・広告なし・外部送信なし（データはこの端末の中だけ）
- 1日に何度でも記録／メモ付き／日付をさかのぼって閲覧・追記
- 記録パレット（アイコンとラベル）は自由に編集
- JSON で書き出し・読み込み（バックアップ／機種変更時の移行）
- オフライン動作（Service Worker）

---

## ファイル構成
```
hitokoma/
├ index.html              アプリ本体（HTML/CSS/JS 全部入り）
├ manifest.webmanifest    PWA 設定
├ sw.js                   オフライン用 Service Worker
├ icon-192.png            アイコン
├ icon-512.png            アイコン
└ icon-maskable-512.png   アイコン（Android 適応型用）
```

## 置き方（どれか1つ）

PWA としてインストール・永続化するには **HTTPS（または localhost）での配信が必須**です。`file://` で直接開くと Service Worker と永続化が動きません。

**A. 自分のサイト（FTP）**
`hitokoma/` フォルダごと HTTPS のサイトにアップロードし、`https://あなたのサイト/hitokoma/` を Android Chrome で開く。

**B. GitHub Pages**
リポジトリに置いて Pages を有効化 → `https://ユーザー名.github.io/リポジトリ名/` を開く。

**C. ローカルで試す（PCで動作確認）**
```
cd hitokoma
python -m http.server 8000
```
→ ブラウザで `http://localhost:8000/`

## ホーム画面に追加（Android）
配信した URL を Chrome で開く → 右上メニュー → 「ホーム画面に追加」。
以降はアイコンから単体アプリのように起動します。

## 使い方
- **記録する**：上のアイコンを押すと、今の時刻でその場で記録されます。メモ欄に書いてからアイコンを押すと、メモも一緒に残ります。
- **修正・削除**：「きょうの糸」の各記録をタップすると、時刻・アイコン・メモの編集と削除ができます。
- **過去の日**：ヘッダーの ‹ › で前後の日へ。日付の文字をタップすると任意の日付へジャンプ。過去日にアイコンを押すと、その日の正午に記録されます（編集で時刻調整可）。
- **バックアップ**：下部「書き出し」で JSON を保存。設定で促す間隔（既定7日）を変えられます。

## バックアップと復元（重要）
データは端末内の IndexedDB に保存されます。アプリの初回起動時に「永続化」を依頼するので、ブラウザに自動削除されにくくなります。ただし**ブラウザのデータ消去・サイトデータ削除・機種変更では消えます**。定期的に「書き出し」で JSON を保存しておいてください。

- **復元／移行**：設定 →「読み込み」で書き出した JSON を選択。
  - OK＝既存に追加（重複 id はスキップ）
  - キャンセル＝全置き換え（現在のデータは消えます）

## カスタマイズ

**名前を変える**：`index.html` の `<title>` と `manifest.webmanifest` の `name`/`short_name`、設定画面の文言を書き換え。

**記録パレット**：アプリの「設定 → 記録パレット」で絵文字とラベルを編集・並べ替え・追加削除できます（コードを触らず変更可）。

**色・テーマ**：`index.html` 冒頭の `:root` と `[data-theme="dark"]` の CSS 変数を変えると配色が変わります。`--accent` がアクセント色です。

**アイコンを差し替える**：`icon-*.png` を同名で置き換え。同梱の `makeicon.py`（Pillow）で再生成もできます。

**アプリを更新したとき**：`sw.js` の `CACHE = "hitokoma-v1"` の版番号を上げると、古いキャッシュが破棄され新版が反映されます。

## 将来の拡張（Google Drive 同期など）
保存処理は `index.html` 内の `Store` オブジェクトに集約してあります。同じメソッド（`addEntry` / `putEntry` / `deleteEntry` / `entriesInRange` / `allEntries` / `getMeta` / `setMeta`）を持つ別プロバイダを用意して差し替えれば、UI を変えずに Google Drive 等へ拡張できます。

## ライセンス / 著作権メモ
このコードはあなたのものとして自由に使えます。アイコン素材も自作なので問題ありません。同種の「アイコンで記録する日記」という発想自体は著作権の対象外（アイデアは保護されない）ですが、他アプリの具体的なアイコン画像・コード・固有の画面デザインの流用は避けてください。
