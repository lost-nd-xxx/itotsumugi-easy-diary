# いとつむぎ — ローカル日記 PWA

一日のかけらを、アイコンで簡易に綴る日記アプリです。
スマホ（Android Chrome）で「ホーム画面に追加」すると、アプリのように使えます。

- 完全無料・広告なし・外部送信なし（データはこの端末の中だけ）
- 1日に何度でも記録／複数アイコン選択／メモ付き
- 過去日にさかのぼって閲覧・追記・編集
- 全日付横断フィード（紡いだ糸）＋ カレンダー表示
- ジャンル・アイコンによる絞り込み
- 記録パレット（アイコンとラベル）は自由に編集
- JSON で書き出し・読み込み（バックアップ／機種変更時の移行）
- オフライン動作（Service Worker）

---

## ファイル構成
```
itotsumugi/
├ index.html              アプリ本体（HTML/CSS/JS 全部入り）
├ manifest.webmanifest    PWA 設定
├ sw.js                   オフライン用 Service Worker
├ icon-192.png            アイコン
├ icon-512.png            アイコン
└ icon-maskable-512.png   アイコン（Android 適応型用）
```

## 置き方（どれか1つ）

PWA としてインストール・永続化するには **HTTPS（または localhost）での配信が必須**です。`file://` で直接開くと Service Worker と永続化が動きません。

**A. 自分のサイト（FTP）**
`itotsumugi/` フォルダごと HTTPS のサイトにアップロードし、`https://あなたのサイト/itotsumugi/` を Android Chrome で開く。

**B. GitHub Pages**
リポジトリに置いて Pages を有効化 → `https://ユーザー名.github.io/リポジトリ名/` を開く。

**C. ローカルで試す（PCで動作確認）**
```
cd itotsumugi
python -m http.server 8000
```
→ ブラウザで `http://localhost:8000/`

## ホーム画面に追加（Android）
配信した URL を Chrome で開く → 右上メニュー → 「ホーム画面に追加」。
以降はアイコンから単体アプリのように起動します。

## 使い方

- **記録する**：上のアイコンを1つ以上選択し、「記録」ボタンを押すと今の時刻で記録されます。複数ジャンルをまたいで選択できます。
- **日時・メモを指定**：「日時・メモ」アコーディオンを開くと、任意の日時とメモを付けて記録できます。
- **修正・削除**：「紡いだ糸」の各記録をタップすると、時刻・アイコン・メモの編集と削除ができます。
- **カレンダー**：月表示で各日のアイコンを確認。日付タップでその日の記録一覧を表示・追記できます。年月ジャンプボタンで任意の月へ飛べます。
- **絞り込み**：ジャンルまたは個別アイコンで絞り込み。フィードとカレンダーの日ビューに反映されます。
- **バックアップ**：設定 →「書き出し（JSON）」で保存。設定で催促間隔（既定7日）を変えられます。

## バックアップと復元（重要）
データは端末内の IndexedDB に保存されます。**ブラウザのデータ消去・サイトデータ削除・機種変更では消えます**。定期的に「書き出し」で JSON を保存してください。

- **復元／移行**：設定 →「読み込み」で書き出した JSON を選択。
  - OK＝既存に追加（重複 id はスキップ）
  - キャンセル＝全置き換え（現在のデータは消えます）

## カスタマイズ

**記録パレット**：設定 →「記録パレット」で絵文字とラベルを編集・並べ替え・追加削除できます（コードを触らず変更可）。

**色・テーマ**：`index.html` 冒頭の `:root` と `[data-theme="dark"]` の CSS 変数を変えると配色が変わります。`--accent` がアクセント色です。

**アイコンを差し替える**：`icon-*.png` を同名で置き換え。同梱の `makeicon.py`（Pillow）で再生成もできます。

**アプリを更新したとき**：`sw.js` の `CACHE = "itotsumugi-v1"` の版番号を上げると、古いキャッシュが破棄され新版が反映されます。

## 将来の拡張（Google Drive 同期など）
保存処理は `index.html` 内の `Store` オブジェクトに集約してあります。同じメソッド（`addEntry` / `putEntry` / `deleteEntry` / `entriesInRange` / `allEntries` / `getMeta` / `setMeta`）を持つ別プロバイダを用意して差し替えれば、UI を変えずに Google Drive 等へ拡張できます。
